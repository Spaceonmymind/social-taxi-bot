📋 Порядок подачи заявки

  • Заявка подаётся минимум за 3 суток до даты поездки.
  • После отправки заявки администратор её рассматривает и подтверждает через систему.
  • Заявка считается принятой только после подтверждения администратором. После подтверждения вам придёт сообщение с данными автомобиля и стоимостью поездки.
  • Поездки выполняются только в рабочие дни согласно графику.
  • Социальное такси работает только по городу Кургану.

🚫 Правила отмены

  • Отмена заявки возможна не позднее чем за 3 часа до времени поездки.
  • Для отмены воспользуйтесь разделом «Мои заявки» или свяжитесь с оператором.
  • Повторное оформление заявки возможно на другую дату.

🪪 Документы для поездки

  • Паспорт или иной документ, удостоверяющий личность.
  • Для инвалидов — справка об инвалидности (МСЭК).
  • Для пенсионеров — пенсионное удостоверение.
  • Для участников СВО и членов их семей — соответствующее удостоверение.
  • Документы предъявляются водителю перед началом поездки.
